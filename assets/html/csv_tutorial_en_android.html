How to build .csv file for import?

CSV file format is very simple: wikipedia
It's a text file that consists of coma (or semicolon) separated fields organised in lines.
Each line should represent one object (library entry), like this:


"Green apple";"100";"g";"46";"12.1";"0.4";"0.4"
"Red apple";"100";"g";"47";"12.1";"0.4";"0.4"


All values must be in quotes ("{content}").


You can have as many lines as you like. The best way would be to prepare input file in spread sheet and export to .csv. Make sure that you wrap all values in "" and that you user semicolon (;) as separator!.
You can also add few items to the library manually and simple export it to see how it looks like:)


Columns are:
1 - name (string)
2 - amount in the described entry (100 by default)
3 - unit of described entry (g - grams by default)
4 - kilocalories in described entry
5 - amount of carbs in described unit in grams
6 - amount of fat in described unit in grams
7 - amount of protein in described unit in grams


File must not have a column row!
